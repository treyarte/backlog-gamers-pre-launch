## Introduction

The Backlog Gamers Pre-Launch Site serves as a teaser and informational platform for my upcoming gaming site, [Backlog Gamers](https://www.backloggamers.com). The site uses Mailchimp to sign up users to a mailing list.

## Features

- **Mailing Sign up**: Signs users up to a mailing list to receive updates, news, and early access notifications.
- **Information**: Learn what Backlog Gamers is with about my section that highlight features about the Backlog Gamers platform.
- **FAQs**: Find answers to frequently asked questions about the Backlog Gamers platform and its features.

## Requirements

- Node Version 18.16.0 or higher

## Getting Started

### Running locally

To get started clone the repository and run the commands below.

```sh
    npm install && npm run dev
```

npm install will install the node modules and npm run dev will start the server. To visit the local site type in localhost:3000 in your browser.

### Building and deploying for production

To run the site in production run the following commands below"

```sh
npm install
npm run build
npm start
```
